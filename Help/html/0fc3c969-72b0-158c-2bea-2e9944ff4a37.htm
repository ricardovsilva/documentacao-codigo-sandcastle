DBRef Class
Native type that maps to a database reference. Use Database.FollowReference(DBRef) to retrieve the document that it refers to.

Remarks
DBRefs are just a specification for a specially formatted Document. At this time the database does no special handling of them. Any referential integrity must be maintained by the application not the database.
